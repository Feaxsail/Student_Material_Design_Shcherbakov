# Student List Application

## Цель работы
Научиться применять принципы Material Design в Android-приложениях с использованием Jetpack Compose.

## Функциональность
- Список студентов с аватарками
- Поддержка светлой и тёмной темы
- Кастомная цветовая схема (зелёная тема)
- Кастомная типографика (шрифты Montserrat и Abril Fatface)
- Анимация раскрытия карточек с описанием студента
- Top App Bar с логотипом

## Технологии
- Kotlin
- Jetpack Compose
- Material Design 3
- Git & GitHub
